# Bounty Briefs

One file per bounty. Each file contains:

- Bounty number, name, tier, RICE score
- Problem statement (plain language)
- "Shipped + Proven" acceptance criteria
- Suggested approach
- Shared frameworks this bounty depends on
- Related bounties
- Known gotchas

Sourced from the master bounty board (v3.2): `msp-launchpad/projects/level-up-event/bounty-board-v3.md` in Lodewijk's workspace.

## Status

Files populated as bounty briefs are finalized. Lodewijk + Thanh own the content.
